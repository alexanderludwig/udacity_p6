## Instructions
  

### Clone project
` git clone https://github.com/alexanderludwig/udacity_p6.git`

`cd udacity_p6`

`npm install`

`ganache-cli`

`truffle compile`  
`truffle migrate`  

In a separate terminal window, launch the DApp:  
`cd src`  
`npm run dev`  

### Deployment  
Deploy to Rinkeby network:  
`compile`  
`truffle migrate --reset --network rinkeby`
